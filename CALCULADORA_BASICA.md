# CALCULADORA BASICA
## ==Operaciones Disponibles==
1. Suma(+)
2. Resta(-)
3. Multiplicación(*)
4. Division(/)
5. Cerrar

**Elige una opción:**

## SUMA(+)
Paso para realizar una suma
1. Ingrese el primer numero
2. Agrega el operador **"+"** 
3. Ingresa el segundo numero
4. Realiza la sumatoria
5. Muestra el resultado final

## RESTA(-)
Paso para realizar una resta
1. Ingresa el primer numero
2. Agrega el operador **"-"** 
3. Ingresa el segundo numero
4. Realiza la diferencia
5. Muestra el resultado final

## MULTIPLICACION(*)
Paso para realizar un producto
1. Ingresa el primer numero
2. Agrega el operador **"*"** 
3. Ingresa el segundo numero
4. Realiza el producto
5. Muestra el resultado final

## DIVISION(/)
Paso para realizar un division
1. Ingresa el primer numero
2. Agrega el operador **"/"** 
3. Ingresa el segundo numero
4. Realiza la division
5. Muestra el resultado final
